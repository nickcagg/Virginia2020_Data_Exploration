# Virginia2020_Data_Exploration

## All data was obtained from `data.census.gov` (see data_cleaning.ipynb for specific links)

All of the data was limited to ZIP codes within the State of VIrginia published for the year 2020

### Overall Objective

Understanding the demographics of the *Commonwealth of Virginia* using K-means clustering and a Gaussian Mixture Model to segment the state with data centered on population (size, age, gender), industry, and education.
